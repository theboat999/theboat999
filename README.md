## Current Focus

**QHSE Management Systems** — Digitizing construction safety workflows with role-based access control, incident reporting, and real-time analytics as part of the Bauer QHSE Management System — a PWA-ready portal replacing paper-based processes entirely.

**IoT & Computer Vision** — Building POMODAERO, an aeroponic farming application integrating YOLOv11 for automated growth stage identification and environmental monitoring.

**LLM & MCP Integration** — Exploring the intersection of local LLM deployment and Model Context Protocol (MCP) for efficient, context-aware tooling in production workflows.
